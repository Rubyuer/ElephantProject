# 小象木梳

  "小象木梳"是一款线上购买木梳的应用。
  
  【类型多】
  小象木梳提供小象·木质、小象·镶齿、小象·角梳和小象·梳镜类商品。
  
  【流程】 
  a.选择心仪梳子-梳子详情-加入购物车,购物车里进行结算;
  b.选择心仪梳子-梳子详情-立即购买,进行单件梳子的购买。

  【名气大】 
  小象木梳APP,是倍受年轻人喜爱的梳理头发的正品购买平台。
  
  【鉴别平台】 
  经过多年行业的调研,对梳子进行专业质检,助用户网购梳子提供优质的商品。

  【退换货】 
  若在小象木梳买到不适合的梳子,小象木梳在此承诺:可支持"无小票、无实物、无理由"直接退换货。
